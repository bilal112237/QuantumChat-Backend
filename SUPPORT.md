# Support

## How to get help

| Need | Where |
| ---- | ----- |
| Backend bugs / API questions | [Issues](https://github.com/QuantumLogicsLabs/QuantumChat-Backend/issues) |
| Client / crypto UI issues | [QuantumChat-Frontend](https://github.com/QuantumLogicsLabs/QuantumChat-Frontend/issues) |
| Product docs | [QuantumChat](https://github.com/QuantumLogicsLabs/QuantumChat) |
| Security reports | [SECURITY.md](SECURITY.md) |

```bash
npm ci
npm run test:security
npm run test:security:x5
```

See [README.md](README.md) for environment variables and CI lanes.

Maintainers respond as capacity allows; security reports are prioritized.
